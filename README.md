マークシートを科目ごとに分類するプログラム．

「分類したい画像たち(jpegファイルなど)が入ったフォルダ」が入ったフォルダと同じところに本プログラムを置き，指示に従ってフォルダ名を入力すれば，
画像をコピーする形で科目ごとに分類してくれる．
